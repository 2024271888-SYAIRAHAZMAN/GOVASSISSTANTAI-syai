# 🏛️ GovAssistant AI
**Empowering Citizens Through Generative AI & High-Speed Logic**

GovAssistant AI is a modern digital gateway designed for the **Project 2030: MyAI Future Hackathon**. 
It simplifies how Malaysian citizens interact with government services by turning complex portal navigation into a simple experience.

## Project Overview
GovAssistant leverages the power of **gemini-3.1-flash-lite-preview** to provide instant, accurate, and polite guidance on government-related inquiries, such as STR assistance, passport renewals, and B40 initiatives.

---

## Key Features:
* **Fully AI-Powered:** No hardcoded scripts; the logic is driven by real-time Generative AI.
* **High-Speed Response:** Optimized using the gemini-3.1-flash-lite-preview model for low-latency interactions.
* **Secure Architecture:** Built with a Node.js backend to ensure API keys and sensitive logic are handled server-side.
* **Citizen-Centric UI:** A clean, patriotic interface designed for accessibility and ease of use.

---

## 🛠️ Tech Stack & Tools
* **Model:** gemini-3.1-flash-lite-preview
* **Runtime:** Node.js & Express
* **Development:** Visual Studio Code
* **Cloud Infrastructure:** Prepared for Google Cloud Run (Containerized via Docker)
* **AI Coding Assistant:** Developed with the help of Gemini for Code to ensure optimized and clean script logic.

---

## 🏗️ Technical Architecture
1.  **Frontend:** HTML, CSS, and JavaScript.
2.  **Backend:** Node.js server acting as a secure bridge.
3.  **API Integration:** Secure REST connection to Google AI Studio.
4.  **Deployment:** Dockerized environment ready for Google Cloud Run's serverless infrastructure.
